My Projects

Here are some of the projects that I created while taking DataViz at Lehigh.

Infographic

This infographic was made using Canva to represent Lehigh University undergraduate enrollment by college. Although this data can be found in many university reports or other written statistics, I think that this graphic is visually pleasing and adds a more exciting visual element to an otherwise mundane set of numbers. The Second infographic was made for my global workplace class to compare the hourly minimum wages of different countries.

Lehigh University Student Division by College 2016

Hourly Minimum Wages 2016

Chart

This chart was created using the application Chartbuilder, a program that enables you to import data from a spreadsheet and then edit the data to create an appropriate chart. The chart that I created represents annual food consumer price index in the United States over a 10-year period.

USA Annual Food Consumer Price Index (2004-2014)

Timeline

This timeline was made using TimelineJS, an application that allows you to use a google sheet template to import excel data and then edit it to create a timeline. I chose to create a timeline of women's fashion trends through the decades since I found this topic interesting and visually appealing in a timeline.

Women's Fashion

Google Fusion Maps

Google Fusion is a program that allows you to create an interactive map with cards that show information from different locations on the map. I chose to create a map of the top five Lehigh Valley locations for people who might be interested in the attractions that the Bethlehem area has to offer.

Top Five Lehigh Valley Locations

Tableau

Tableau is an application that allows you to create interactive visuals based on a dataset. Using TableauPublic, you can publish your visual to the web for viewers to see. For my visual, I created an interactive map of median household incomes in the United States.

USA Median Household Income 2015
